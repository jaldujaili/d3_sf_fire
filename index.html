Project Theme

We would like to present some stories regarding fire safety in San Francisco in 2018. We can find interesting patterns with the time series data. Moreover, we can find interesting geographical information with the dataset since we are provided the Longitude and Latitude for each record. We would be able to tell from this dataset that what types of incidents happened in San Francisco as well.

Data and Processing

Data source: Fire Department Calls for Service, San Francisco (Our dataset was downloaded on 02/18/2019)

Attribution Value Sreenshot

Data Types String, Number, Date, Datetime, Boolean, Location (Latitude and Longitude)

Data Processing: Filtered "City" column to remove data that's not happened in "San Francisco". Filtered "Call Date" column to remove data that's not happened in "2018". Removed redundant columns, such as "Call Number", "Unit ID", "Incident Number", "Location", "RowID". See the script source code in "data" folder.
